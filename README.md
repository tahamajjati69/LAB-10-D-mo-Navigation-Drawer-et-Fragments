# LAB 10 – Navigation Drawer et Fragments
## Objectif
### Ce TP a pour objectif de créer une application Android utilisant :

* Navigation Drawer
* Fragments
* FragmentManager
* Menu latéral
* Navigation dynamique entre plusieurs écrans



# Technologies utilisées
* Android Studio
* Java
* XML
* Android SDK
* Navigation Drawer
* Fragments
# Structure du projet
* MainActivity.java
* BlankFragment.java
* BlankFragment2.java
* FragmentList.java
* activity_main.xml
* fragment_blank.xml
* fragment_blank2.xml
* activity_main_drawer.xml
